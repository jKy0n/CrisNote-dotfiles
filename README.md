# Operational System:
- OS: Gentoo Linux 2.17 x86_64
- Kernel: 6.6.X-gentoo-dist
- Shell: zsh (bash for sh)
- GS: XOrg (X11)
- WM: awesome
- Audio: pipewire
- Terminal: Alacritty + tmux

- FS: BtrFS (system + home)
- Init: systemd
